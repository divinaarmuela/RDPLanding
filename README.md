# Real Deal Property — Free Training Landing Page

A single static HTML page (`index.html`). No build step, no dependencies.

## Deploy with GitHub Pages
1. Create a new GitHub repo (e.g. `rdp-landing`).
2. Add `index.html` to the repo root and commit.
3. Repo → **Settings → Pages** → Source: **Deploy from a branch** → Branch: `main` / `/root` → **Save**.
4. Live in a minute or two at `https://<username>.github.io/<repo>/`.

### Custom domain (optional)
Settings → Pages → Custom domain → enter e.g. `invest.realdealproperty.com.au`, then add a `CNAME` record at your DNS pointing to `<username>.github.io`.

## Before it goes live
- **Wire the form to your CRM (HubSpot).** Right now it only shows a confirmation message. Point it at your form endpoint so it captures the lead and triggers the video + email/SMS flows.
- **Add the video.** Embed or link the recorded training where the opt-in leads.
- **Fill the green placeholders:** Google rating + review count, the comparison dollar figures, and real client reviews (with consent).
- **Verify the proof points** (600+, $150M+, 10%+, 12 years, 7%+) before publishing.
